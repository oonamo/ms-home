# Manage My Home
## Setup
Install lua using these [Instructions](http://lua-users.org/wiki/BuildingLuaInWindowsForNewbies)

```
# in root directory
make
```
Config path is located in `C:\Users\user\AppData\Local\ms_home\conf.lua`
